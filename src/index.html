Deploy to GitHub Pages
  1. Create a new repository on GitHub (public) and name it however you like.
  2. Upload the generated portfolio HTML file as index.html to the repository root (or push via git).
  3. Go to Settings → Pages, select Branch: main (root), Save. Your site will be published at https://<username>.github.io/<repo>/

Tip: For a user site at https://<username>.github.io use repo named <username>.github.io and push index.html to main.
